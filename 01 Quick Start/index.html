Open Layers Map


        1. OpenLayers `Map` obj를 만들기.
          var map = new ol.map({...});
        2. div의 id 값인 'map'을 'target'의 인자로 지도객체를 연결.
          target : 'map'
        3. `layers` : [...] 맵의 가능한 layer 목록을 정의하는 데 사용함. 
            소스는 맵 타일을 얻는데 사용되는 프로토콜
          layers: [
            new ol.layer.Tile({
              source: new ol.source.OSM()
            })
          ]
        4. `view`에서는 지도의 중심, 해상도, 회전을 저장할 수 있음.
            center 지정된 좌표체계는 (ESPG:4326)임
